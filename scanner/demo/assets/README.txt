ZeroTrace – Demo: Plattform-Logos
=================================

Lege hier deine eigenen Logo-Dateien ab, damit die Demo sie auf dem
Auswahl-Screen anzeigt. Erwartete Dateinamen (relativ zur ZeroTrace-Demo.html):

  assets/fivem.png    -> FiveM
  assets/altv.png     -> alt:V
  assets/ragemp.png   -> RageMP

Empfehlung: quadratische PNGs mit Transparenz (z. B. 128x128). Fehlt eine
Datei, zeigt die Kachel automatisch ein schlichtes Ersatz-Symbol; der Name
darunter (FiveM / alt:V / RageMP) ist immer sichtbar.

Hinweis: Die Logos sind Marken der jeweiligen Anbieter. Verwende sie nur,
wenn du dazu berechtigt bist.
